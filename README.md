# 🌍 Global CO₂ Emission Forecasting (Time Series ML Project)

This project focuses on analyzing and forecasting global CO₂ emissions using historical data and machine learning techniques.

## 📊 Dataset

* **Year** (Decadal intervals + recent yearly data)
* **Emissions** (Million Metric Tons)

## 🎯 Objectives

* Understand historical emission trends
* Analyze growth patterns and percentage changes
* Forecast future CO₂ emissions up to **2060**

## 🧠 Methodology

* Data Cleaning & Preprocessing
* Feature Engineering (Lag features, growth rates)
* Log Transformation for stability
* Exploratory Data Analysis (EDA)

## 🤖 Models Used

* Linear Regression
* Random Forest Regression

## 🔮 Forecasting Approach

* Recursive forecasting using lag-based features
* Dynamic updating of inputs for future predictions

## 📈 Key Highlights

* Time series converted into supervised learning problem
* Strong trend modeling using log transformation
* Comparative analysis of linear vs non-linear models
* Visualization of actual vs predicted emissions
* Percent change analysis for deeper insights

## 🚀 Future Enhancements

* Implement **Direct Forecasting (multi-horizon models)**
* Build a **hybrid model (Linear + Tree-based residual learning)**
* Improve feature set using rolling statistics and seasonality
* Add interactive dashboards for visualization (e.g., Plotly/Streamlit)

## 📌 Outcome

The project provides a structured approach to forecasting long-term CO₂ emissions and demonstrates practical time series modeling using machine learning.
